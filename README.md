# A.Kaushik Portfolio

A modern and responsive developer portfolio built using Next.js, React, Tailwind CSS, and Framer Motion. This portfolio showcases my projects, technical skills, and contact information through a clean and professional user interface.

---

## 🚀 Features

- Modern & Responsive UI
- Smooth Scrolling Navigation
- Animated Sections with Framer Motion
- Projects Showcase
- Skills & Technologies Section
- Resume Download Option
- GitHub Integration
- Contact Section
- Optimized for Desktop & Mobile Devices

⚡ Installation & Setup

Clone the repository:

git clone https://github.com/Akaushik3003/kaushik-portfolio.git

Navigate into the project folder:

cd kaushik-portfolio

Install dependencies:

npm install

Run the development server:

npm run dev
